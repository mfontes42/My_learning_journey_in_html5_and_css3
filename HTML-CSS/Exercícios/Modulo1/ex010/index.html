Listas em HTML

Listas ordenadas (ordered lists - ol)

Vamos criar uma lista ordenada usando a tag <ol>:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

Listas nao ordenadas (unordered lists - ul)

Vamos agora criar uma lista nao ordenada usando a tag <ul>:

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Listas mistas

Tambem podemos criar listas mistas configurando listas dentro de outras listas:

  1. Item 1
  2. Item 2
  3. Item 3
    • Item 3.1
    • Item 3.2
    • Item 3.3
  4. Item 4
  5. Item 5
    1. Item 5.a
    2. Item 5.b
    3. Item 5.c

Listas de definicao (definition lists - dl)

Existe ainda um tipo de lista que e a de definicao que funciona como um dicionario, na verdade, vejamos abaixo usando a tag <dt>:

HTML
Linguagem de marcacao para criacao de sites.
CSS
Linguagem de marcacao para a criacao do design de um site.
JavaScript
Linguagem de programacao para a criacao de interatividade de um site.
